While only admins will have access to the 8-digit door security code, the code can be shared with other organization members or outside security to run the door. They will simply need to enter the code on their profile tab to access the scanner.

### 1. Toggle to the Events tab 
### 2. Tap the event to open the event admin page
### 3. Tap the 'Door Security' button and tap on the 8 digit code to copy to clipboard
### 4. Exit the event admin page and toggle to the profile tab
### 5. Tap the 'Door Security' button and enter or paste the 8 digit code
### 6. Scan guests DoorList QR code or tap full list to see a full list of guests (hold down on their name to enter them)
