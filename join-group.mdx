## Direct Add

Admins of a group can add you directly via invitation or roster upload

## Join Link/Code

Potential members can join groups via link or join code

- **Link:** You may request to join the group or freely join the group depending on Must Request To Join settings
- **Join Code:** You may instantly join a group if you have the Join Code by inserting the Join Code in the Profile tab of the app (under Join Group) or on doorlist.live
