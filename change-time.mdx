--- 
title: "Change Event Date and Time"
description: "How to change the date and time of an event on DoorList"
---

## 1. Toggle to the Calendar tab 
## 2. Tap the event to open the event admin page
## 3. Tap the menu icon in the tap right corner of the event admin page
## 4. Tap the edit event button 
## 5. Select a new date and time
## 6. Tap Save in the upper right corner
