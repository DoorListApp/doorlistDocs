---
title: "Duplicate an Event List"
description: "How to create a new event with the same list a previous event"
---

## 1. Toggle to the Calendar tab 
## 2. Tap the event to open the event admin page
## 3. Tap the menu icon in the tap right corner of the event admin page
## 4. Tap the duplicate event button
## 5. Give your new event a name and start time
## 6. Tap Duplicate
