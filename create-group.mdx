title: 'Create a Group'
description: 'Learn how to create a group on DoorList'
---

## Method 1

1. Tap the Blue + Icon in the middle of the dock

2. Tap 'Create Group'

3. Name your group

4. Provide a description (*optional)*

5. Select a School

6. Select Visibility

   1. Private (*default*): Only admins can add members to the group

   2. Request: Users must be approved by an admin to join the group

   3. Public: Anyone can join the group

## Method 2

1. Toggle to the home tab and select tap 'Add a Group'

2. Select 'New Group'

3. Name your group

4. Provide a description (*optional)*

5. Select a School

6. Select Visibility

   1. Private (*default*): Only admins can add members to the group

   2. Request: Users must be approved by an admin to join the group

   3. Public: Anyone can join the group

*If you are the first group at your school, you will need to create a new school*
